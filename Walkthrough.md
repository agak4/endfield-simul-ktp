# Walkthrough - Endfield Calculator

## 구현된 기능
- **오퍼레이터 데이터**: 20명의 오퍼레이터 스킬, 재능, 잠재 능력 데이터 구현 (`data_operators.js`)
- **무기/장비 데이터**: 주요 무기 및 장비 세트 효과 구현
- **데미지 계산 엔진**: 
  - 기본 데미지 공식 (공격력 * 배율 * 각종 증폭/취약/저항)
  - 스킬 시퀀스 데미지 계산 (사이클 DPS)
  - 각종 버프/디버프 및 스택(Generic Special Stack) 처리 로직
  - UI:
    - 오퍼레이터/무기/장비 선택 및 스탯 표시
    - 스킬 시퀀스 편집 (드래그 앤 드롭)
    - 상세 데미지 리포트 (스킬별 비중, 버프 로그)
    - 툴팁 시스템 (스킬/특성 상세 설명 및 실시간 적용 효과 표시)

## 최근 수정 사항
- **Bug Fix**: `js/calc.js`의 `evaluateTrigger` 함수에 `strictMode` 파라미터를 추가.
  - 무기/장비 효과(패시브)는 `strictMode=false`로 처리하여, 스택이 0이어도 해당 능력을 보유한 오퍼레이터라면 효과가 발동되도록 수정 (예: 장대한 염원).
  - 스킬/궁극기 보너스(추가 데미지)는 `strictMode=true`로 처리하여, 실제 스택이 존재해야만 발동되도록 수정 (예: 관리자 궁극기).
- **UI Fix**: `js/ui/tooltip.js`에서도 스킬 툴팁 표시 시 `strictMode=true`를 적용하여 보너스 표시 로직을 일관성 있게 유지.
